USING TABLES IN MS ACCESS

    INTRODUCTION

  • Microsoft Access is a Database Management System (DBMS), a part of the Microsoft Office Suite.

  • MS Access is one of the most flexible and easy-to-use databases that allows user to create many applications without writing a single line of the code.

  • For something very powerful MS Access provides a powerful programming language i.e. Visual Basic for Applications (VBA)
    •

    NAVIGATION PANE

  • The main control center of each Access database is the Navigation Pane. It shows you what is in the database and makes the information accessible to you.

  • Also, you can choose to close the navigation pane, if you need more of the Access window to complete specific tasks.

    •

    UNDERSTANDING DATABASE OBJECTS

  • TABLES - Tables are the fundamental objects in Access and are used to hold the raw data of the database.

  • FORMS - Forms can be described as the 'face' of an application. They are customized screens that provide an essay way to enter, view and modify data.

  • QUERIES - Queries are requested to extract relevant information from a table or tables. Queries can be based on table(s) or other queries. They can be used to select, change, add or delete records in the database.

  • REPORTS - A Report is a properly presented and explanatory listing of the data printed on a paper or viewed on the screen.
