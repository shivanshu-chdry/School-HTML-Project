DATABASE MANAGEMENT WITH MS ACCESS

    TYPES OF DATABASE

  • There are two types of databases that can be used in real-world scenarios

  • Flat-File databases are generally plain text files that can be used by local applications to store data.

  • Relational database are databases that store the data in several tables and link those tables to access meaningful information.
