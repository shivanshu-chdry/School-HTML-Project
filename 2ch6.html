MS ACCESS: FORMS, QUERIES & REPORTS

    CREATING QUERIES

  • You can create query either from scratch using design view or by using a wizard.

  • In Design View, a query lets you enter criteria to specify which records of a table are to be displayed.

  • In Datesheet view of query, all the fiels and records in a table that you had selected while designing a query will be displayed.

    UNDERSTANDING REPORTS

  • It is a properly presented and explanatory listing of data, printed on paper.

  • Report is an effective way of presenting data in a printed format because you have control over the size and appearence of everything in a report, therefore you can display information in the way you want to see.
