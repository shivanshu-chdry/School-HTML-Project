NETWORKING CONCEPTS

    TYPES OF NETWORK CONFIGURATIONS

  • PEER-TO-PEER NETWORKS - Peer-to-peer (P2P) networks are more commonly implemented where less then ten computers are involved and where strict security is not necessary.

  • CLIENT/SERVER NETWORKS - Client/server networks are more suitable for larger networks.

    •

    COMPONENTS OF A NETWORK

  • SERVERS - Servers are faster computers that run various software, store and process information by providing an interface for the users to be able to utilize the networked computers.

  • NODES - Nodes are the computers on the network that are provided to the users in order to carry out their tasks using the network.

  • NETWORK INTERFACE CARD (NIC) - Network card is a necessary component of a computer without which a computer cannot be connected over a network.

  • HUBS / SWITCHES - When you connect multiple computers on a network, they all plug into a central device called a hub whereas Switches are really smart hubs such that they are able to build tables to keep track of which computer is on which switch port.

  • NETWORK OPERATING SYSTEM (NOS) - The network requires some software to control all the information transfer activity on the network, like the traffic police controls the traffic.

  • NETWORK CABLES - This is the medium or channel over which the information travels from computer to computer.
