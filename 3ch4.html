DATABASE MANAGEMENT WITH MS ACCESS

    UNDERSTANDING THE CONCEPT OF DATABASE MANAGEMENT SYSTEM

  • Database management system (DBMS) is a software package that provides software programs to control the creation, maintenance and use of a database.

  • The diagram given shows the database systems and explanation of its divisions.

  • A File in a Database can be defined as a collection of related records.

  • A Record is the detail or related information of an individual

  • A field is an individual data item within a record like an address field or name field etc.

  • A database is a collection of files that act as a single centralized data bank for many processing applications.

    UNDERSTANDING MS ACCESS

  • Microsoft Access is a part of MS Office suite and an information management tool that helps you store information for reference, reporting and analysis.

  • It can be used for various purposes, from maintaining a simple list to implementing a full-featured data management tool to storing large quantities of information and much more.

    •

    WHAT IS MS ACCESS?

  • MS Access is a Relational Database Management System (RDBMS)

  • A relational database means a database that allows you to group its data into one or more distinct tables that can be related to one another by using field or fields common to each other.
