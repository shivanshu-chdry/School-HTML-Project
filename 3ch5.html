USING TABLES IN MS ACCESS

    FIELD NAMES

  • The field name column specifies some name for the field. The name can be practically anything desired by you, with an exception of not repeating names in the same table.

    •

    DATA TYPES

  • Data Type column, which is next to the Field Name column, specifies the type of information stored in the field. The edit box for this column is a drop down list box. The list contains all the available data types.

    •

    DESCRIPTION

  • A short description about the field can be entered in this column but it is optional.

    •

    FIELD PROPERTIES

  • Each field in a table has its own set of properties and they further define the field and its utility in the database. The available properties change, depending on the data type chosen for a field.
