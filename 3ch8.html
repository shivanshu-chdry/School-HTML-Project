    EXITING PYTHON

    BASIC OPERATORS IN PYTHON

  • + >> Addition >> x + y

  • - >> Subtraction >> x - y

  • * >> Multiplication >> x * y

  • ** >> Exponent >> x ** y

  • / >> Division (float) >> x / y

  • // >> Division (floor) >> x // y

  • % >> Modulus >> x % y
