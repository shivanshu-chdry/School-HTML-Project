NETWORKING CONCEPTS

    NETWORK TOPOLOGIES

  • BUS TOPOLOGY - In the bus network topology, every workstation is connected to a main cable calld bus.

  • RING TOPOLOGY - It is called ring topology, because it forms a ring wherein each node is connected to another computer, with the last one connected to the first.

  • STAR TOPOLOGY - In the star network topology, there is central computer or server or hub to which all the workstations are directly connected. Every workstation is indirectly connected to every other through the central computer.

  • TREE TOPOLOGY - Tree topology also called hierarchical topology has a root node and all other nodes are connected to it forming a hierarchy. It should at least have three levels of hierarchy.

  • MESH TOPOLOGY - The mesh network topology employs either of two schemes, called full mesh and partial mesh.
