PYTHON REVIEW

    PYTHON KEYWORDS

  • Python keywords are reserved words in Python that should not be used as variables, constants, function names, or identifiers in your code.

  • These keywords have a special meaning and they are used for special purposes in Python programming language.

  • There are total 33 keywords in Python 3.10. To get the keywords list at the >>> prompt type help() and press Enter key.

  • Type keywords to get the list of the keywords for the current pythn version running on your operating system.

    VARIABLES AND DATA TYPES IN PYTHON

  • Variables are used to store data, they take memory space based on the type of value we assign to them.

  • You can create the variables by writing the variable name on the left side of = and the value on the right side.
    For example - my_variable = 10


    • VARIABLE NAME (IDENTIFIERS)

  • Variable name is known as identifier.

  • The name of a variable can start with either a letter or an underscore(_) with a limit of 79 characters.

  • The name of the variable cannot start with a number.

  • The name of the variable cannot have special characters such as %, $, # etc.

  • Variable name is case sensitive in Python.
