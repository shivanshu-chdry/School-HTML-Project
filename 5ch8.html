PYTHON REVIEW

    PYTHON DATA TYPES

  • Data types defines the type of the variable, whether it is an integer variable, string variable, list, etc.

  • A variable can hold different types of values.

  • The data types defined in Python are given below:
  • Numbers
  • String
  • List
  • Tuple
  • Dictionary

    •

    NUMBERS

  • Number data types store numeric values. Python creates Number objects when a number is assigned to a variable.

  • Python supports four types of numeric data:
  • int
  • long
  • float
  • complex

  • We can use the type() function to know which class a variable or a value belongs to.
