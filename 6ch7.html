CREATING WEB SITES USING HTML

    LINKING WEB PAGES

  • <a> ANCHOR TAG - HTML uses the <a> (anchor) tag to create a link to another document. With this, one can make clickable links between pages. But <a> tag by itself does not do anything. While creating links to other pages or location, <a> tag includes the full Internet address or file name of the other pages.

    HOW TO SEND THE LINK BACK?

  • If you want to send the link back from feature.html to ubuntu.html. Use the following command in the features.html

    • <a href= "../ubuntu.html"> Return to the Home Page </a>

    Here the (..) double dots indicate to go back to the previous/folder page.

    alink and vlink ATTRIBUTES

  • alink is used to set the color for the active links in a web page.

  • vlink attribute is used to set the color for the visited link in a web page.

  • Both these are the attributes of the <body> tag.
