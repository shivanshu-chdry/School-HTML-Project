PYTHON REVIEW

    STRING

  • String is a sequence of characters in Python.

  • The data type of String in Python is also called 'str'.

  • They are either enclosed my single quotes(' ') or double quotes(" ")

  • The plus sign is the string concatenation operator and asterisk (*) is the repetition operator.

    •

    PYTHON COMMENTS

  • A comment is the text that doesn't affect the output of a code.

  • It is just a piece of text to let someone know what you have done in a program.

  • There are two types of comments in Python:
  • Single line comment
  • Multiple line comment

  • In python we use #(hash sign) which is a special character to start the comment.

  • To have a multi-line comment we use triple single quotes(''') at the beginning and end of the comment
