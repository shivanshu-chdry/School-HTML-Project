PYTHON REVIEW

    input() FUNCTION

  • Python input() function is used to get input from the user.

  • If the input function is called, the program flow will be stopped until the user has entered the input and then ended the input with the return key.
    •

    eval() FUNCTION

  • The eval function is used to evaluate valid Python expressions which are stored in a string.

  • Python provides the eval function that attempts to evaluate a string in the same way that the interactive shell would evaluate it.
